NP_ModComments
==============

NP_ModComments.php is a fork of the ModComments plugin for NucleusCMS which I (Matt) feel could be better. To use it add tot he plugins folder of your installation of NucleusCMS.

The aim of this project is to createa  really awesome meta-moderation system for Nucleus CMS

The version of ModComments should work with NP_Memberscore: http://wakka.xiffy.nl/memberscore
